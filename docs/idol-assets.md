# Idol Asset Guide

把透明背景立绘放到这个目录：

`public/assets/idols/`

## Naming Rule

- Format: `.webp` preferred, `.png` also supported
- Background: transparent
- Suggested height: `1600px` to `2200px`
- Safe area: keep the character centered; avoid cropping hair accessories and hands
- File name must exactly match the idol id below

## Fallback Behavior

- If the `.webp` file is missing, the site will try `.png`
- If both are missing, the UI shows a stylized placeholder block automatically
