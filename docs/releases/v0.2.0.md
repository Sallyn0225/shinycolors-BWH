# v0.2.0

## Summary

This release makes the project easier to publish and easier to understand.

- Added ready-to-use deployment config for Vercel, Netlify, and Cloudflare Pages
- Refreshed the README to work better as a GitHub landing page
- Added license and copyright files with clear scope boundaries
- Updated maintenance docs to explain the SPA deployment behavior

## Highlights

### Multi-platform deployment

- Added `vercel.json`
- Added `netlify.toml`
- Added `public/_redirects`
- Kept `BrowserRouter` and solved refresh fallback at the platform layer

### Better GitHub presentation

- Reworked the README with badges, deployment guidance, and a cleaner structure
- Added a release note document for reuse in GitHub Releases
- Clarified where to start for local development and maintenance

### Licensing clarity

- Added an MIT `LICENSE` for original code and docs
- Added `COPYRIGHT` to separate original repo work from third-party IP and assets
